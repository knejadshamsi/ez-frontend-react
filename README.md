# EZ Frontend React

A React-based frontend application with map visualization capabilities.

## Features

- Map visualization using Mapbox GL and deck.gl
- Zone selection
- Layer control and switching
- TypeScript support
- State management with Zustand

## Getting Started

### Installation

```bash
yarn install
```

### Development

```bash
yarn start
```

Runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build

```bash
yarn build
```

Builds the app for production to the `build` folder.

## Technologies

- React 18+
- TypeScript
- Mapbox GL
- deck.gl
- Ant Design
- Zustand
- Axios
